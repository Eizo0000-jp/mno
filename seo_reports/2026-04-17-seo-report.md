## 🔍 SEO改善ポイント

### 1. 🕷️ インデックス・クロール環境の整備（最優先）

Search Consoleにクエリが**1件も表示されていない**のは、Googleがサイトをほとんどクロール・インデックスできていないサインです。

**具体的なアクション：**
```
✅ Google Search Console で「URL検査」→ インデックス登録をリクエスト
✅ sitemap.xml を作成・送信（例: /sitemap.xml）
✅ robots.txt を確認（誤ってクロール拒否していないか）
✅ Google Search Console の「カバレッジ」レポートでエラーを確認
```

---

### 2. 📄 コンテンツの絶対量不足の解消

現在ブログ記事は**2本のみ**。Googleが「このサイトは何について書いているサイトか」を判断できる情報量に達していません。

**目標：**
```
📌 最低でも月4〜8本のペースで記事を投稿
📌 「楽天モバイル 社員紹介」関連の検索意図を網羅する
📌 1記事あたり1,500文字以上を目安に
```

---

### 3. 🎯 ターゲットキーワードの明確化と内部構造の整備

現状、サイトのテーマが**「社員紹介キャンペーン」なのか「楽天モバイル全般の情報サイト」なのか**がGoogleに伝わっていない可能性があります。

**具体的なアクション：**
```
✅ TOPページのtitleタグ・meta descriptionを
   「楽天モバイル 社員紹介 キャンペーン」軸で最適化

✅ 例）
   title: 楽uten モバイル社員紹介キャンペーン完全ガイド｜mobile-friend.com
   description: 楽天モバイルの社員紹介キャンペーンを使ってお得に申し込む方法を解説。
                紹介コードの受け取り方・特典・注意点まで徹底解説。

✅ パンくずリスト・カテゴリ構造を整備してサイト全体の
   トピッククラスター（関連記事の束）を形成する
```

---

### 4. ⏱️ 直帰率100%・滞在2秒の改善（UX/CTA改善）

TOPページの滞在時間が**2秒**は、ページが表示される前に離脱しているレベルです。

**チェックリスト：**
```
✅ ページ表示速度の確認（PageSpeed Insights で計測）
✅ モバイル表示の最適化（スマホユーザーが大半のはず）
✅ ファーストビューに「誰向けのサイトか」「何ができるか」を
   明確に記載（例：「楽天モバイルをお得に始めるなら社員紹介が最強」）
✅ 紹介コード申込みへの導線（CTA）をファーストビューに配置
```

---

### 5. 🔗 外部からの被リンク・流入経路の開拓

現状、オーガニック検索もほぼなく、SNS流入も2セッションのみ。サイトへの**入口**が存在しない状態です。

**具体的なアクション：**
```
✅ X（Twitter）・Instagramで記事をシェアする習慣をつける
✅ 楽天関連のコミュニティ（Reddit的なもの・知恵袋）で
   自然な形で情報提供 → サイトへ誘導
✅ 他の楽天モバイル系ブログ・サイトとの相互リンクを検討
✅ Googleビジネスプロフィールへの登録（補助的に）
```
